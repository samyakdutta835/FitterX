🏋 Back Exercises

Build a wider, thicker and stronger back with the most effective exercises.

Pull Ups

Advanced

Target Muscle : Lats and Upper Back

Sets : 4

Reps : 8-12

Common Mistakes

  • Not using full range of motion.
  • Swinging the body.
  • Partial repetitions.

Lat Pulldown

Beginner

Target Muscle : Lats

Sets : 4

Reps : 10-12

Common Mistakes

  • Pulling behind the neck.
  • Using momentum.
  • Leaning excessively.

Barbell Row

Intermediate

Target Muscle : Middle Back

Sets : 4

Reps : 8-12

Common Mistakes

  • Rounded lower back.
  • Jerking the weight.
  • Standing too upright.

Deadlift

Advanced

Target Muscle : Entire Posterior Chain

Sets : 4

Reps : 5-8

Common Mistakes

  • Rounding the lower back.
  • Starting with hips too high.
  • Pulling with the arms.

T-Bar Row

Intermediate

Target Muscle : Middle Back

Sets : 4

Reps : 8-12

Common Mistakes

  • Using momentum.
  • Rounded back.
  • Incomplete contractions.

Dumbbell Row

Beginner

Target Muscle : Lats and Middle Back

Sets : 4

Reps : 10-12

Common Mistakes

  • Twisting the torso.
  • Using too much weight.
  • Short range of motion.

Hyperextensions

Beginner

Target Muscle : Lower Back

Sets : 3

Reps : 12-15

Common Mistakes

  • Overextending the spine.
  • Using momentum.
  • Fast repetitions.

Chin Ups

Advanced

Target Muscle : Lats and Biceps

Sets : 4

Reps : 8-12

Common Mistakes

  • Partial repetitions.
  • Swinging the body.
  • Not controlling the descent.
